# Сессионный проект по курсу “Операционные системы семейства Unix”

## Описание

В репозитории собраны артефакты по заданию по Unix/Fedora 43.

Среда выполнения:

- macOS Apple Silicon
- виртуальная машина Fedora 43 aarch64 в UTM
- сеть локальной подсети 192.168.1.0/24
- статический IP виртуальной машины: 192.168.1.100
- hostname: mephi-2026.domain.local

## Состав артефактов

- project_history.txt — история команд
- network_check.txt — проверка сетевой связности
- nginx_recent_logs.txt — журнал сервиса nginx
- fstab.txt — конфигурация автомонтирования
- selinux_status.txt — текущий режим SELinux
- file_contexts.txt — SELinux-контекст web-каталога
- tcpdump_capabilities.txt — capabilities для tcpdump
- permissions.txt — владелец и права каталога /data/mephi-web
- users_groups.txt — сведения о пользователе и группе
- index.html — опубликованная web-страница
- curl_output.txt — результат обращения к web-серверу по IP
- mephi-nginx-screenshot.png — скриншот результата
- tcpdump-4.99.6-2.fc43.aarch64.rpm — локально скачанный RPM-пакет

### SELinux

Файл: selinux_status.txt
Enforcing
Файл: file_contexts.txt

Показывает, что для каталога /data/mephi-web установлен тип httpd_sys_content_t, необходимый для работы nginx с нестандартной директорией.

### Права доступа

Файл: permissions.txt

Подтверждает, что каталог /data/mephi-web принадлежит пользователю mephi-admin и группе mephi-devs, а права установлены как 2775, включая setgid-бит.

Файл: users_groups.txt

Содержит:
- информацию о пользователе mephi-admin
- информацию о группе mephi-devs
- подтверждение, что пользователь mephi-admin добавлен в группу mephi-devs

### Настройка capabilities для tcpdump

Файл: tcpdump_capabilities.txt

Подтверждает, что для /usr/sbin/tcpdump установлены capabilities:

- cap_net_raw
- cap_net_admin

Это позволяет запускать tcpdump обычному пользователю без SUID.
